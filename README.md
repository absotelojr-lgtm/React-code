# Bitcoin Mining Dashboard

A React-based dashboard that simulates Bitcoin mining activity, visualizes mining statistics, and provides a modern UI. No real mining is performed; this app demonstrates mining concepts and stats.

## Features
- Simulated mining statistics (hashrate, blocks found, earnings)
- Start/stop mining simulation
- Real-time log/activity viewer
- Mining settings/configuration section
- Modern UI with Material-UI

## Getting Started

1. **Install dependencies:**
	```bash
	npm install
	```
2. **Start the app:**
	```bash
	npm start
	```
3. Open [http://localhost:3000](http://localhost:3000) to view it in your browser.

## Project Structure
- `src/App.js` — Main app structure and navigation
- `src/components/Dashboard.js` — Mining stats and controls
- `src/components/Settings.js` — Mining configuration
- `src/components/LogViewer.js` — Real-time log display

---
This project is for educational/demo purposes only and does not perform real Bitcoin mining.
